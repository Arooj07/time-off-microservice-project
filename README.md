# Time-Off Microservice

A NestJS-based microservice that manages employee time-off requests with external HCM synchronization.

## Tech Stack

- NestJS
- SQLite
- TypeORM
- Jest (testing)

## Features

- Time-off request lifecycle
- Balance management per employee & location
- HCM integration (mocked)
- Batch & real-time sync
- Defensive error handling

## Run Project

npm install
npm run start

## Run Tests

npm run test

## API Prefix

/api
